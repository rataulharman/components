# Car Gallery React App

This is a simple React project created as part of my coursework.  
It showcases a collection of my favorite cars using a clean layout.

## Features

- Header with navigation links  
- Banner section with buttons  
- Dynamic gallery using `.map()`   
- Footer with basic information  


## Built With

- React  
- JavaScript (ES6)  
- HTML & CSS  


## Project Structure

- `App.js` – Main component  
- `Header.js` – Navigation bar  
- `Banner.js` – Intro section  
- `Button.js` – Reusable button  
- `Gallery.js` – Displays car images  
- `Footer.js` – Footer section
